# Christmas Web Page 🎄

인터랙티브한 크리스마스 웹페이지입니다.

## 기능

- **눈 내리는 효과** - 화면에 눈송이가 내립니다. 아무 곳이나 클릭하면 눈송이가 추가됩니다.
- **크리스마스 트리** - 반짝이는 조명과 장식이 달린 트리
- **마법 버튼** - 클릭하면 폭죽이 터집니다
- **선물 상자** - 클릭하면 깜짝 메시지가 나타납니다
- **카운트다운** - 다음 크리스마스까지 남은 시간 표시

## 실행 방법

`index.html` 파일을 브라우저에서 열면 됩니다.

```bash
open index.html
```

## 기술 스택

- HTML5
- CSS3 (애니메이션, Flexbox)
- Vanilla JavaScript
